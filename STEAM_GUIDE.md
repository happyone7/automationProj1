# Steam 출시 가이드

## 📋 출시 전 필수 조건

### 1. Unity 빌드 생성
```
File → Build Settings → Windows → Switch Platform
↓
Build And Run → 폴더 선택
```

### 2. 빌드 파일 확인
- exe 파일 존재 확인
- 데이터 폴더 포함 확인

---

## 🚀 Steam Direct 출시 프로세스

### Step 1: Steamworks 계정 생성
```
1. https://partner.steamgames.com 접속
2. Steam 계정으로 로그인
3. 이메일 인증
4. 사업자 정보 입력 (개인 개발자 가능)
```

### Step 2: Steam Direct fee 지불
- **비용**: $100 (1회)
- **환급**: 누적 판매 $1,000 달성 시 환급
- 결제 수단: PayPal 또는 카드

### Step 3: 새 앱 생성
```
Steamworks → 새로운 제품 추가 → 게임/소프트웨어 선택
↓
앱 이름: Automation Game (예시)
↓
앱 ID 발급 (나중에 사용)
```

### Step 4: 스토어 페이지 작성

| 섹션 | 내용 |
|------|------|
| **게임 이름** | Automation Factory |
| **short description** | 1-2문장 게임 소개 |
| **full description** | 상세 게임 설명, 특징 |
| **특징** | 자동화,_factory, 인디 |
| **태그** | Automation, Factory, Idle, Strategy |
| **릴리스 날짜** | 출시 예정일 |
| **가격** | $4.99 또는 무료 |

### 스크린샷/트레일러 준비
- **최소**: 5장 스크린샷 (1280x720 이상)
- **권장**: 트레일러 영상 (MP4, 최대 500MB)
- ** Capsule 이미지**: 460x215 (자동 생성됨)

### Step 5: 빌드 업로드

**방법 1: Steamworks SDK 사용**
```
1. SDK 다운로드 (Steamworks → 도구)
2. SteamPipe CLI 사용
3. Depot 생성 → Content 폴더 업로드
```

**방법 2: 타사 도구 (초보 권장)**
- **SteamPipe GUI**: 무료, 사용简单
- **GameJolt Upload**: 대안

### Step 6: 검토 프로세스
- **기간**: 1-5일 (평균 2-3일)
- **검토 내용**:
  - 실행 파일 동작 확인
  - 부적절한 콘텐츠 확인
  - 필수 요소 (설명, 스크린샷)

### Step 7: 출시!

| 출시 유형 | 설명 |
|----------|------|
| **즉시 출시** | 승인 즉시 공개 |
| **예약 구매** | 특정 날짜에 공개 |

## 💰 수익 구조

| 판매액 | 개발자 수령 | Steam |
|--------|------------|-------|
| $0 - $10M | 70% | 30% |
| $10M - $50M | 75% | 25% |
| $50M+ | 80% | 20% |

**예시**: $4.99 게임
- 개발자: $3.49
- Steam: $1.50
